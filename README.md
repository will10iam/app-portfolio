## Para usar

Quer ter uma cópia local?  - Siga os passos abaixos:

### Pré-requisitos

- npm
  ```sh
  npm install npm@latest -g
  ```

### Instalação

1. Clone the repo
   ```sh
   git clone https://github.com/will10iam/app-portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
